## About

[ClickHouse](https://clickhouse.com) ([GitHub](https://github.com/ClickHouse/ClickHouse)) database destination for [Fivetran](https://fivetran.com) automated data movement platform, based on the [Fivetran Partner SDK](https://github.com/fivetran/fivetran_sdk).
